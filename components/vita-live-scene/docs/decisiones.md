# Decisiones aprobadas

- Utilizar exactamente las seis fotografías de pacientes proporcionadas.
- Los pacientes no serán clicables.
- VITA permanece visualmente estática.
- El objetivo principal es comunicar atención práctica y directa mediante WhatsApp.
- La composición será asimétrica y editorial, no circular.
- Desktop y mobile tendrán adaptaciones propias.
- La escena debe sentirse activa, humana y premium.
- No repetir logos de WhatsApp como decoración.
- Usar microelementos ambientales discretos.
- Respetar `prefers-reduced-motion`.
- Nombre oficial único: `vita-live-scene`.
